# 1. Node.js

1.JavaScript runtime environment.
2.Runs JavaScript outside the browser.
3.Required to run React applications.

4.Verify installation:

node -v
npm -v

# 2. React Installation

1.Open VS Code Terminal.

2.Go to the desired folder:

3.cd folder_name

4.Create React project:

npx create-react-app my-app

5.Open project:

cd my-app
code .

6.Run project:

npm start

# 3. Linking React with VS Code

1.Install VS Code.
2.Install Node.js.

3.Open project using:

code .
Open Terminal.

4.Run:

npm start
React opens in the browser.

# 4. Important Files

src/ → React source code.
public/ → Static files.
node_modules/ → Installed packages.
package.json → Project information and dependencies.

node -v          # Check Node.js version
