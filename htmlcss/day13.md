# 1_Node.js

    Node.js is a JavaScript runtime environment that allows JavaScript code to run outside the web browser.It is used to build server-side applications and is required to create and run React projects.

    Installation Steps:-
    1.Download Node.js from the official website.
    2.Install it using the default settings.
    3.Verify installation using:-

        node -v
        npm -v
        (npm is installed automatically with Node.js.)

# 2_React Installation

    React is a JavaScript library used for building fast,interactive,and reusable user interfaces(UI),especially for single-page applications.

    Installation steps:-
    1.Open VS Code Terminal.

    2.Navigate to the desired folder:-
        cd folder_name

    3.Create a React project:-
        npx create-react-app my-app

    4.Open the project:-
        cd my -app
        code .

    5.Start the React application:-
        npm start

# 3_Linking React with VS Code

    VS Code is used as the code editor to write,edit,and run React applications.After creating a React project,open it in VS Code to begin development.

    Steps:-
    1.Install VS Code
    2.Install Node.js
    3.Open the React project folder in VS Code using:-
        code .
    4.Open the integrated terminal.
    5.Run:-
        npm start

    6.The React application will open automatically in the default web browser.
