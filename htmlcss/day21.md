# User Login and Todo Management

User Login and Todo Management is a web application that allows users to register, log in securely, and manage their personal tasks. After logging in, users can add, view, update, and delete their own todos. The application ensures that each user can access only their own data, providing privacy and security. User information and todo details are stored in a PostgreSQL database, while the frontend and backend communicate through REST APIs. This system helps users organize their daily tasks in a simple and efficient way.

Key Features:-

1.User Registration and Login
2.Secure Authentication
3.Add, View, Update, and Delete Todos
4.User-specific Todo Management
5.PostgreSQL Database Integration
6.Simple and User-friendly Interface..
